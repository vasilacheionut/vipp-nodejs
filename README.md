# My nodejs web-scraper
This is app web-scraper 
Load data from
https://www.nike.com/it/launch?s=upcoming

Open terminal:
- git clone https://github.com/vasilacheionut/vipp-nodejs.git
- cd vipp-nodejs
- npm install

Npm global install
- sudo npm install forever -g

Run app
- node index.js
- npm start
- npm stop


Test App:
- http://localhost:3000/nike/arrivo
